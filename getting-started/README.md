# Getting Started
This is very beginnig of learning ***Go*lang**.

## hello-world.go
- Go program  
  - an extension `.go`
  - first line must be declaring *package*
  - `func` to define a function
  - `main()` is entry point for Go program
- 

hello-world.go
```
package main

import "fmt"

func main() {
	fmt.Println("Hello, World!")
}
```
